# Protable-Mobile-Charger-Using-Arduino
Peripheral Project

### Equipments
1. Arduino.
2. Breadboard.
2. LED.
3. 9v battery.
4. Battery cap.
5. Jumper wires.
6. 7805 ic.
7. 100 kOhm, 1 MOhm Registers.
8. Potentiometer.
9. LCD Display.
10.Key-Pad.
